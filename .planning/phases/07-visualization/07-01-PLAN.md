- ringSpacing from smoothedSize: 8 + smoothedSize * 12 (8-20px spacing)
- ringCount from currentFingerprint.ringCount
- Source position from smoothedProx: sourceDistance = 30 + (1 - smoothedProx) * 80, offset left of center
- For each ring: compute radius with i-based spacing * breathScale (with staggered phase offset: breathPhase + i * 0.3f for ripple-out effect), skip if > maxRadius
- Apply shape distortion: xR = radius * currentFingerprint.xStretch, yR = radius * currentFingerprint.yStretch
- Apply angular warp: for asymmetry > 0, modify ellipse control points using angularWarp * sin(angle) perturbation
- Opacity: blend between direct alpha (higher for inner rings, scaled by proximity) and reflected alpha (higher for outer rings, scaled by 1-proximity), multiplied by layerAlpha
- Skip rings with alpha < 0.02f
- Draw as Path::addEllipse with AffineTransform::translation, stroke with AetherColours::inkLight at computed alpha, 0.8px stroke

- After the final output trim section (at the end of processBlock, after OutputSection::process), add RMS computation:
  ```cpp
  // RMS level for visualization breathing boost
  {
      float rms = buffer.getRMSLevel (0, 0, buffer.getNumSamples());
      if (getTotalNumInputChannels() > 1)
          rms = std::max (rms, buffer.getRMSLevel (1, 0, buffer.getNumSamples()));

      // Keep peak since last GUI read (compare-and-swap)
      float expected = visualizationRmsLevel.load (std::memory_order_relaxed);
      while (rms > expected &&
             !visualizationRmsLevel.compare_exchange_weak (expected, rms,
                 std::memory_order_relaxed))
      {}
  }
  ```

This uses compare-and-swap to retain the peak value between GUI reads. The visualization calls `exchange(0.0f)` to consume and reset.
