  <action>
    Create a `ParchmentElements` namespace with static drawing functions. All functions take a `juce::Graphics&` parameter and draw at the specified coordinates.

    **Function 1: `generateBackground(int width, int height)` -> `juce::Image`**
    Pre-renders the full background (parchment + texture + vignette) to a cached Image:
    1. Create `juce::Image(juce::Image::ARGB, width, height, true)`
    2. Create Graphics from image
    3. Fill with Parchment colour (0xfff0e6d3)
    4. Draw paper texture: horizontal lines every 4px across full height. Colour: Sepia (0xff704214) at alpha 0.018f (within the 0.015-0.02 range from LOCKED DECISION). Use `g.setColour(Colour(sepia).withAlpha(0.018f))` then `g.fillRect(0, y, width, 1)` in a loop.
    5. Draw edge vignette: Radial ColourGradient from transparent at centre (width/2, height/2) to Sepia at 0.08f alpha at corners. Use radial gradient with `addColour(0.6, Colours::transparentBlack)` so darkening starts at 60% from centre, reaching ~30px inward visible effect (LOCKED DECISION). The endpoint should be at corner distance: `sqrt((width/2)^2 + (height/2)^2)`.
    6. Return the Image.

    **Function 2: `drawDiamondDivider(Graphics& g, float y, float leftX, float rightX)`**
    Draws a horizontal divider with a diamond at the centre (LOCKED DECISION: functional separator):
    1. Calculate midX = (leftX + rightX) / 2
    2. Set colour to Ink Ghost
    3. Draw left line from leftX to midX - 8px at y, 1px stroke
    4. Draw diamond: Create a juce::Path rectangle (-2.5, -2.5, 5, 5) rotated 45 degrees and translated to (midX, y). Stroke with 1px PathStrokeType.
    5. Draw right line from midX + 8px to rightX at y, 1px stroke

    **Function 3: `drawCornerBrackets(Graphics& g, juce::Rectangle<int> bounds)`**
    Draws L-shaped brackets at all four corners of the given bounds (LOCKED DECISION: moderate/balanced weight):
    1. Bracket size: 20x20px, positioned 6px from edges, 1px Ink Ghost stroke
    2. Top-left: horizontal line from (6, 6) to (26, 6), vertical line from (6, 6) to (6, 26)
    3. Top-right: horizontal line from (width-26, 6) to (width-6, 6), vertical line from (width-6, 6) to (width-6, 26)
    4. Bottom-left: horizontal line from (6, height-6) to (26, height-6), vertical line from (6, height-26) to (6, height-6)
    5. Bottom-right: horizontal line from (width-26, height-6) to (width-6, height-6), vertical line from (width-6, height-26) to (width-6, height-6)
    6. Use 1px line thickness with Ink Ghost colour

    **Function 4: `drawDoubleRule(Graphics& g, float y, float leftX, float rightX, bool isTop)`**
    Draws Victorian double-rule border (LOCKED DECISION: two distinct visible lines, 3px gap):
    1. Primary line: 1px at y, full Ink Ghost colour, from leftX to rightX
    2. Secondary line: 1px at y + 3px (if isTop, below primary) or y - 3px (if bottom, above primary), at 40% opacity of Ink Ghost
    3. Use `g.drawLine(leftX, y, rightX, y, 1.0f)` for each line

    **Function 5: `drawLetterSpacedText(Graphics& g, const juce::String& text, float x, float y, float width, float tracking, const juce::Font& font, juce::Colour colour, juce::Justification justification)`**
    Draws text with custom letter spacing (JUCE has no built-in letter-spacing):
    1. Use juce::GlyphArrangement to measure total text width with tracking
    2. For each character in text:
       a. Create a GlyphArrangement for single character
       b. Measure its width
       c. Accumulate total width including tracking between characters
    3. Calculate starting X based on justification and total measured width
    4. Draw each character at its position:
       a. Create GlyphArrangement, addLineOfText for single char
       b. Draw at current X position
       c. Advance X by char width + tracking pixels
    5. Set colour before drawing

    Alternatively (simpler, Crucible-proven approach): Use `GlyphArrangement::addLineOfText()` for the full string, then iterate glyphs and offset each glyph's x position by `glyphIndex * tracking`. Then call `glyphs.draw(g)`.

    **Function 6: `drawSectionLabel(Graphics& g, const juce::String& numeral, const juce::String& name, float x, float y, float width, const juce::Font& numeralFont, const juce::Font& nameFont)`**
    Draws "I. Cabinet Resonance" style section labels (LOCKED DECISION: equal visual weight for numeral and name):
    1. Draw numeral (e.g., "I.") using numeralFont (Cormorant Garamond 11px italic) in Ink Faint colour
    2. Draw section name using nameFont (Spectral 9px uppercase, 3px tracking) in Ink Faint colour
    3. Position name after numeral with 4px gap
    4. Use drawLetterSpacedText for the section name to get the 3px tracking
