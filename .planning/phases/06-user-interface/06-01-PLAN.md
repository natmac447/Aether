    **AetherLookAndFeel.h** -- Class declaration:
    - Inherits `juce::LookAndFeel_V4`
    - Three `juce::Typeface::Ptr` members: displayTypeface (Cormorant Garamond), bodyTypeface (EB Garamond), labelTypeface (Spectral)
    - Public font helpers: `getDisplayFont(float size)`, `getBodyFont(float size)`, `getLabelFont(float size)` -- each returns Font with typeface or Georgia fallback
    - Override declarations: drawRotarySlider, drawButtonBackground, drawButtonText, drawLabel, drawPopupMenuBackground, drawPopupMenuItem, getTextButtonFont, getLabelFont(Label&), getPopupMenuFont

    **AetherLookAndFeel.cpp** -- Implementation:

    **Constructor:**
    - Load all 3 typefaces from BinaryData using `Typeface::createSystemTypefaceFor()`. Use the Regular weight for each (Cormorant Garamond Regular, EB Garamond Regular, Spectral Regular). Also load Light and Italic variants as additional members if needed for specific typography roles.
    - Null-check each typeface after loading. If null, leave as nullptr (font helpers will fallback).
    - Set JUCE colour IDs for standard components to match Aether palette (Label::textColourId = Ink, TextButton colours, ComboBox colours, PopupMenu colours, etc.)

    **Font helpers:**
    - `getDisplayFont(size)`: If displayTypeface not null, return `juce::Font(juce::FontOptions(displayTypeface).withHeight(size))`. Else return `juce::Font(juce::FontOptions("Georgia", size, juce::Font::plain))`.
    - Same pattern for getBodyFont (bodyTypeface), getLabelFont (labelTypeface).
    - Store italic variants as separate typeface pointers. Provide `getDisplayFontItalic(size)`, `getBodyFontItalic(size)`, `getLabelFontItalic(size)`.

    **drawRotarySlider override:**
    - Calculate bounds, centre, radius (with 2px padding for shadow)
    - Draw drop shadow: 1px offset, Colour(AetherColours::shadow), 3px blur radius
    - Draw body: Radial ColourGradient from Parchment Light (top-left offset by -0.4*radius) to Parchment Dark (bottom-right offset by +0.4*radius). Fill ellipse.
    - Draw inner highlight: 0.3 opacity white, inset 1px from top
    - Draw border: 1.5px Ink Light ellipse stroke
    - Draw indicator: Calculate angle from sliderPos mapped to startAngle..endAngle. Draw 1.5px wide line from (centre + innerR*sin, centre - innerR*cos) to (centre + outerR*sin, centre - outerR*cos) where innerR = radius - 16px and outerR = radius - 2px. Use Ink colour.
    - Arc range: 270 degrees (7 o'clock start = pi*1.25, 5 o'clock end = pi*2.75, clockwise)

    **drawButtonBackground override:**
    - For toggle buttons (getToggleState): Active = Ink fill with 2px corner radius. Inactive = transparent with 1px Ink Ghost border.
    - For bypass buttons (check component name or property): Active(In) = Ink Light border. Inactive(Out) = Ink Ghost border at 60% opacity.
    - For other buttons: Parchment background with Ink Ghost border.

    **drawButtonText override:**
    - For toggle active: Parchment colour text (ink-inverted)
    - For toggle inactive: Ink Faint text
    - For bypass buttons: use Spectral 7px uppercase with 2px tracking. Active = Ink Light, Bypassed = Ink Ghost.
    - Use getLabelFont for button text.

    **drawLabel override:**
    - Draw text in label's text colour with getLabelFont. Handle centred/left/right justification.

    **drawPopupMenuBackground override:**
    - Fill with Parchment. Draw 1px Ink Ghost border.

    **drawPopupMenuItem override:**
    - Highlighted: Accent Warm background, Parchment text.
    - Normal: Parchment background, Ink text.
    - Use getBodyFont for menu items.

    **getTextButtonFont, getLabelFont(Label&), getPopupMenuFont:**
    - Return appropriate font (getLabelFont for buttons, getBodyFont for labels, getBodyFont for popup menu).

    Important: Do NOT set this as the default LookAndFeel. It will be set on the editor instance in Plan 04.
