**DiffuseTailSection.h -- Complete rewrite of the stub:**

Include signalsmith headers (delay.h, filters.h, mix.h) and `<array>`, `<cmath>`.

Class constants:
```cpp
static constexpr int kNumFDNLines = 8;
static constexpr int kNumDiffusionStages = 4;
```

FDN base delay times in ms (logarithmically distributed 22-63ms):
```cpp
static constexpr float kFDNDelayMs[kNumFDNLines] = {
    22.0f, 26.5f, 31.5f, 37.0f, 43.5f, 50.0f, 56.5f, 63.0f
};
```

Input diffusion delay times in ms:
```cpp
static constexpr float kDiffusionDelayMs[kNumDiffusionStages] = {
    1.5f, 3.0f, 5.0f, 8.0f
};
```

Class members:
- **Pre-delay:** `signalsmith::delay::Delay<float> preDelayL, preDelayR;`
- **Input diffusion (per channel, 4 stages):** `signalsmith::delay::Delay<float> diffDelayL[4], diffDelayR[4];` (explicit delay+feedback allpass, NOT biquad -- research recommends this for 1.5-8ms delays)
- **FDN delay lines:** `signalsmith::delay::Delay<float> fdnDelayLines[kNumFDNLines];`
- **FDN damping lowpass (per line):** `signalsmith::filters::BiquadStatic<float> dampingFilters[kNumFDNLines];`
- **FDN state:** `float fdnDelaySamples[kNumFDNLines] = {};`, `float feedbackGains[kNumFDNLines] = {};`
- **Diffusion state:** `float diffDelaySamplesL[4] = {};`, `float diffDelaySamplesR[4] = {};` (slightly different L/R for decorrelation)
- **SmoothedValue members:** `decaySmoothed` (20ms ramp), `diffusionSmoothed` (20ms ramp), `preDelaySmoothed` (50ms ramp), `bypassBlend` (10ms ramp)
- **HF damping cutoff:** `float dampingCutoffHz = 12000.0f;` (set by Air link)
- **Shape influence:** `float shapeDiffusionBias = 0.3f;`, `float shapeModalCharacter = 0.2f;` (set by shape link)
- **Current state:** `double currentSampleRate`, `int currentBlockSize`, `bool bypassed`

Public methods: `prepare`, `process`, `reset`, `setBypass`, `setDecay`, `setDiffusion`, `setPreDelay` (receives Room Size normalized value), `setHFDamping` (receives Air normalized value), `setShapeInfluence` (receives shape index for tail character influence).

Private: `configureDelayLengths()`, `updateDecay(float decayMs)`, `updateDampingFilters()`, `nearestPrime(int)` (copy from ResonanceSection or extract to shared utility).

**DiffuseTailSection.cpp -- Full implementation:**

1. **nearestPrime()** -- Same implementation as ResonanceSection (copy the static helper).

2. **configureDelayLengths():**
   - For each of 8 lines: convert kFDNDelayMs[i] to samples at current sample rate, quantize to nearest prime
   - Ensure all 8 are distinct primes (bump duplicates upward, same pattern as ResonanceSection)
   - Store in fdnDelaySamples[]
   - For input diffusion: convert kDiffusionDelayMs to samples for L and R channels
   - R channel diffusion delays offset by +10-15% for stereo decorrelation:
     `diffDelaySamplesR[i] = diffDelaySamplesL[i] * 1.12f` (prime-quantized separately)

3. **prepare():**
   - Store sample rate and block size
   - Pre-delay: size for max 30ms (matches max Room Size ER delay * 0.7 + margin): `maxPreDelay = ceil(35.0 * sampleRate / 1000.0)`
   - FDN delay lines: size for max 70ms (max delay + margin): `maxFDN = ceil(70.0 * sampleRate / 1000.0)`
   - Diffusion delays: size for max 10ms: `maxDiff = ceil(10.0 * sampleRate / 1000.0)`
   - Resize all delay lines, reset all filters
   - Call `configureDelayLengths()`
   - Initialize SmoothedValues: decay 20ms, diffusion 20ms, preDelay 50ms, bypassBlend 10ms
   - Set initial decay to 150ms, diffusion to 0.6, preDelay to corresponding room size default
   - Call `updateDecay(150.0f)` and `updateDampingFilters()`

4. **updateDecay(float decayMs):**
   ```cpp
   float decaySec = decayMs / 1000.0f;
   for (int i = 0; i < kNumFDNLines; ++i)
   {
       float gain = std::pow(10.0f, -3.0f * fdnDelaySamples[i] / (decaySec * (float)currentSampleRate));
       feedbackGains[i] = juce::jlimit(0.0f, 0.9999f, gain);
   }
   ```

5. **updateDampingFilters():**
   ```cpp
   float cutoff = juce::jlimit(1000.0f, 16000.0f, dampingCutoffHz);
   float scaledFreq = cutoff / (float)currentSampleRate;
   for (int i = 0; i < kNumFDNLines; ++i)
       dampingFilters[i].lowpass(scaledFreq);
   ```

6. **process() -- Per-sample stereo processing:**
   ```
   For each sample s:
     a. Read smoothed values: preDelaySamp, diffCoeff (0.0 to 0.7 mapped from 0-1 diffusion), blend
     b. Read stereo input
     c. Write to pre-delay buffers; read from pre-delay at preDelaySamp samples back
     d. Input allpass diffusion (4 stages, per channel):
        For each stage:
          delayed = diffDelayL[stage].read(diffDelaySamplesL[stage])
          output = -diffCoeff * input + delayed + diffCoeff * (stored feedback for this stage)
          diffDelayL[stage].write(input + diffCoeff * delayed)
          input = output  // cascade
        (Same for R channel with diffDelaySamplesR)
     e. Distribute stereo to 8 FDN channels:
        Simple rotation-based upmix (energy-preserving):
        fdnInput[0] = (L + R) * 0.5f  (and distribute with phase offsets)
        OR use manual even/odd distribution:
          Even lines (0,2,4,6) get L, odd lines (1,3,5,7) get R
          Scale by 1/sqrt(4) = 0.5 for energy preservation
     f. FDN loop (read -> damp -> Hadamard -> feedback -> write):
        // Read from delay lines
        float outputs[8];
        for (int i = 0; i < 8; ++i)
            outputs[i] = fdnDelayLines[i].read(fdnDelaySamples[i]);

        // Apply damping lowpass
        for (int i = 0; i < 8; ++i)
            outputs[i] = dampingFilters[i](outputs[i]);

        // Apply Hadamard mixing matrix
        signalsmith::mix::Hadamard<float, 8>::inPlace(outputs);

        // Apply feedback gain and add input, write back to delays
        for (int i = 0; i < 8; ++i)
        {
            float in = outputs[i] * feedbackGains[i] + fdnInput[i];
            fdnDelayLines[i].write(in);
        }
     g. Downmix 8 channels back to stereo:
        // Even lines -> L, odd lines -> R
        float wetL = 0, wetR = 0;
        for (int i = 0; i < 8; i += 2) wetL += outputs[i];
        for (int i = 1; i < 8; i += 2) wetR += outputs[i];
        wetL *= 0.5f;  wetR *= 0.5f;  // normalize
     h. Safety hard limit: jlimit(-4.0f, 4.0f, wetL/wetR)
     i. Add wet tail to input (the tail is additive, not replacing):
        outputL = inputL + wetL
        outputR = inputR + wetR
     j. Apply bypass crossfade:
        finalL = origInputL * (1.0 - blend) + outputL * blend
        finalR = origInputR * (1.0 - blend) + outputR * blend
     k. Write output
   ```

   **IMPORTANT:** The Diffuse Tail is ADDITIVE -- it adds reverb to the signal, not replacing it. The dry signal passes through. Bypass controls whether the wet reverb is added.

7. **Shape influence on tail character** (Claude's Discretion):
   When setShapeInfluence() is called with a shape index, look up the shape's `tailDiffusionBias` and `tailModalCharacter` from the presets. Use tailDiffusionBias to bias the effective diffusion coefficient (add up to +0.15 to the base diffusion value). Use tailModalCharacter to subtly modulate the FDN delay lengths by up to +/-5% -- higher values cluster delay lengths slightly, lower values keep them evenly spread. This makes cathedral tail sound different from studio tail.

8. **setDecay(float decayMs):** Map through SmoothedValue, on each process call check if smoothing and call updateDecay with current value.

9. **setDiffusion(float diffNorm):** Map 0-1 to diffusion coefficient range 0.0-0.7 via SmoothedValue.

10. **setPreDelay(float roomSizeNorm):**
    ```cpp
    // Compressed curve: 70% of ER delay time, capped at 25ms
    float erDelayMs = 1.0f + roomSizeNorm * 29.0f;  // Same mapping as ReflectionsSection
    float preDelayMs = erDelayMs * 0.7f;
    preDelayMs = juce::jlimit(0.5f, 25.0f, preDelayMs);
    float preDelaySamples = preDelayMs * (float)currentSampleRate / 1000.0f;
    preDelaySmoothed.setTargetValue(preDelaySamples);
    ```

11. **setHFDamping(float airAmount):**
    ```cpp
    // Air 0%: bright (12kHz), Air 100%: dark (2kHz)
    dampingCutoffHz = 12000.0f * std::pow(2000.0f / 12000.0f, airAmount);
    dampingCutoffHz = juce::jlimit(1000.0f, 16000.0f, dampingCutoffHz);
    updateDampingFilters();
    ```

12. **reset():** Clear all delay lines, reset all filters, reset SmoothedValues.

**Key design decisions:**
- FDN is additive (adds reverb on top of signal), not replacing the input
- Pre-delay, input diffusion, then FDN (serial chain)
- Stereo via even/odd line distribution (simpler than StereoMultiMixer, sufficient for this use case since the Hadamard matrix provides cross-coupling)
- Safety: hard limit output at +/-4.0f, feedback gains clamped to 0.9999, NaN check in debug builds with jassert
