```cpp
void ReflectionsSection::setAirDarkening (float darkening)

}
```

Modify the existing baseCutoff calculation in process() to subtract air darkening. Find the line:
```cpp
float baseCutoff = 12000.0f - roomSize * 6000.0f;
```
And change it to:
```cpp
float airCutoffReduction = airDarkeningFactor * 3000.0f;
float baseCutoff = 12000.0f - roomSize * 6000.0f - airCutoffReduction;
baseCutoff = juce::jmax (2000.0f, baseCutoff);
```
The jmax floor at 2000Hz prevents the cutoff from going too low (which would make reflections inaudible). The airDarkeningFactor ranges from 0 (no air) to ~1.2 (Warm at 100%), so the max additional reduction is ~3600Hz.

Modify the existing `updateDecay(float decayMs)` method to apply the character bias. Find the line where decayMs is used to calculate feedback gains and apply the bias multiplicatively:
```cpp
float biasedDecayMs = decayMs * (1.0f + characterDecayBias);
```

Use biasedDecayMs instead of decayMs for the feedback gain calculation. The bias is clamped to +/-10%, so at maximum Warm character (tailDecayBias=+0.15 * airAmount=1.0 = +0.15, clamped to +0.10), the decay extends by 10%. At maximum Cold (tailDecayBias=-0.1 * airAmount=1.0 = -0.10), decay tightens by 10%.
