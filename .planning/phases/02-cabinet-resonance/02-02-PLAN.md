    2. prepare() implementation:
       - Store sampleRate and blockSize
       - For each delay line: `delayLines[i].resize(maxDelaySamples + 1)` where maxDelaySamples = ceil(5.0 * sampleRate / 1000.0)
       - `delayLines[i].reset()`
       - Reset all filter states: `bandpassFilters[i].reset()`, `dampingFilters[i].reset()`
       - `lowShelfFilter.reset()`
       - `weightSmoothed.reset(sampleRate, 0.020)` (20ms ramp for Weight)
       - `bypassBlend.reset(sampleRate, 0.010)` (10ms bypass crossfade)
       - `bypassBlend.setCurrentAndTargetValue(bypassed ? 0.0f : 1.0f)`
       - Call `configureForMaterial(kMaterials[currentMaterialIndex])` to set initial filter coefficients

    3. configureForMaterial() implementation:
       - Convert delay times from ms to samples: `delaySamples = delayMs * sampleRate / 1000.0`
       - For each delay line: find nearest prime sample count via nearestPrime() to ensure coprime lengths
       - Store computed delay times in `currentDelayTimeSamples[i]`
       - Configure bandpass filters: `bandpassFilters[i].bandpassQ(bandpassFreqHz / sampleRate, bandpassQ)`
       - Configure damping filters: `dampingFilters[i].lowpass(dampingFreqHz / sampleRate)`
       - Configure low shelf: `lowShelfFilter.lowShelfDb(lowShelfFreqHz / sampleRate, lowShelfGainDb)`
       - Store feedbackGain and outputGain

    4. nearestPrime() static helper:
       - Simple function: starting from target, check upward for primality
       - Ensure the 4 delay line sample counts are all different primes (no shared factors)

    5. process() implementation -- THE CORE DSP LOOP:
       ```
       For each sample in the buffer:
         a. Sum stereo to mono: mono = (L + R) * 0.5f
         b. Get smoothed values: weight = weightSmoothed.getNextValue(), blend = bypassBlend.getNextValue()
         c. Apply quadratic response curve: wetGain = weight * weight
         d. Compute FDN:
            - Read from each delay line: delayOutputs[i] = delayLines[i].read(currentDelayTimeSamples[i])
            - Apply damping lowpass to each output: delayOutputs[i] = dampingFilters[i](delayOutputs[i])
            - Apply Householder matrix: signalsmith::mix::Householder<float, kNumDelayLines>::inPlace(delayOutputs)
            - For each line: input = bandpassFilters[i](mono) + delayOutputs[i] * currentFeedbackGain
            - Write to delay: delayLines[i].write(input)
            - Sum outputs: fdnOut += delayOutputs[i]
            - Normalize: fdnOut *= 0.25f (divide by kNumDelayLines)
         e. Apply output low shelf: fdnOut = lowShelfFilter(fdnOut)
         f. Apply output gain: fdnOut *= currentOutputGain
         g. Blend with bypass: output = mono * (1.0f - blend * wetGain) + fdnOut * blend * wetGain
         h. Write mono output to both L and R channels (CAB-05: mono processing)
       ```
       IMPORTANT: When both weightSmoothed and bypassBlend are not smoothing AND (bypassed OR weight is 0), skip FDN processing entirely for CPU savings. Just pass buffer through unchanged.

    6. setWeight(): `weightSmoothed.setTargetValue(juce::jlimit(0.0f, 1.0f, weight));`

    7. setMaterial(): If materialIndex differs from currentMaterialIndex, update currentMaterialIndex and call configureForMaterial(). Note: For simplicity in this first implementation, filter coefficient changes happen immediately (not crossfaded). The short FDN settles quickly (~5ms) so artifacts from coefficient switching are minimal. Material crossfade can be added as a refinement if needed.

    8. setBypass(): `bypassed = shouldBypass; bypassBlend.setTargetValue(shouldBypass ? 0.0f : 1.0f);`

    9. reset(): Reset all delay lines, filters, SmoothedValues, feedbackState.
