**AirSection.cpp complete implementation:**

`prepare()`:
- Store sampleRate and blockSize
- Initialize airSmoothed with 25ms ramp (sampleRate * 0.025 samples), reset to 0.4 (matching default Air Amount)
- Initialize bypassBlend with 10ms ramp, reset to 1.0 (not bypassed = active)
- Set currentCharIndex = 1 (Neutral default), pendingCharIndex = -1
- charCrossfadeStep = 1.0f / (0.030f * sampleRate) -- 30ms character crossfade
- currentAirSmoothed = 0.4f
- Call updateFilters() to set initial coefficients
- Reset all filter states (call reset on each BiquadStatic)

`reset()`:
- Reset all BiquadStatic filters (call `.reset()` on each hfShelf, lfShelf, allpass for both L/R and both current/pending sets)
- Reset SmoothedValues

`setBypass(bool)`:
- Set bypassBlend target to 1.0f (active) or 0.0f (bypassed)

`setAmount(float airAmount)`:
- Set airSmoothed target to juce::jlimit(0.0f, 1.0f, airAmount)

`setCharacter(int index)`:
- If index == currentCharIndex (or == pendingCharIndex during crossfade), return early
- If not already crossfading: set pendingCharIndex = index, charCrossfade = 1.0f
- If already crossfading (pendingCharIndex >= 0): snap current to pending immediately (currentCharIndex = pendingCharIndex), then set pendingCharIndex = index, charCrossfade = 1.0f
- Call updatePendingFilters() after setting pending
- Reset the pending filter state arrays (clear biquad state for clean crossfade)

`updateFilters()`:
- Get character preset from kCharacterPresets[currentCharIndex]
- Calculate HF shelf: `shelfDb = preset.minShelfDb + currentAirSmoothed * (preset.maxShelfDb - preset.minShelfDb)`. Apply via `hfShelfL.highShelfDb(preset.shelfFreqHz / sr, shelfDb, preset.shelfOctaves)`, same for R.
- Calculate LF shelf (if preset.lfShelfFreqHz > 0): `lfDb = preset.lfMinShelfDb + currentAirSmoothed * (preset.lfMaxShelfDb - preset.lfMinShelfDb)`. Apply via `lfShelfL.lowShelfDb(preset.lfShelfFreqHz / sr, lfDb)`, same for R.
- Calculate allpass Q: `q = preset.allpassMinQ + currentAirSmoothed * (preset.allpassMaxQ - preset.allpassMinQ)`
- For each allpass stage i (0 to preset.allpassStages-1): compute center freq logarithmically spaced from allpassBaseFreqHz over allpassSpreadOctaves. Apply `allpassL[i].allpassQ(freq / sr, q)`. For R channel, offset freq by *1.12f for stereo decorrelation: `allpassR[i].allpassQ(freq * 1.12f / sr, q)`.

`updatePendingFilters()`:
- Same as updateFilters() but operates on pending filter arrays using kCharacterPresets[pendingCharIndex]

`process(juce::AudioBuffer<float>& buffer)`:
- Guard: if numSamples < 1 or numChannels < 1, return
- Early exit optimization: if bypassBlend is not smoothing AND target <= 0.0f, return (fully bypassed)
- Get write pointers for L and R channels
- Check if airSmoothed.isSmoothing() to know if we need per-sample coefficient updates
- Per-sample loop:
  - `float blend = bypassBlend.getNextValue();`
  - If airSmoothing: update currentAirSmoothed from airSmoothed.getNextValue(). Every 16 samples (`(s & 15) == 0`), call updateFilters() and (if crossfading) updatePendingFilters()
  - Store input samples for bypass blend
  - **Process through current character filters:**
    - HF shelf: `outL = hfShelfL(inL)`, `outR = hfShelfR(inR)`
    - LF shelf (if active for current character): `outL = lfShelfL(outL)`, `outR = lfShelfR(outR)`
    - Allpass cascade (for currentCharIndex stages): `outL = allpassL[i](outL)`, `outR = allpassR[i](outR)`
  - **If crossfading (pendingCharIndex >= 0):**
    - Process input through pending character filters (same chain structure but using pending filters)
    - Advance crossfade: `charCrossfade -= charCrossfadeStep`
    - Blend: use `signalsmith::mix::cheapEnergyCrossfade(charCrossfade, currentGain, pendingGain)`. Output = currentOut * currentGain + pendingOut * pendingGain
    - If charCrossfade <= 0.0f: complete crossfade -- swap pending to current (currentCharIndex = pendingCharIndex, pendingCharIndex = -1, charCrossfade = 1.0f), copy pending filter states to current filter arrays (or just call updateFilters to reconfigure current filters with the new character)
  - **Bypass blend:** `finalL = inL * (1.0f - blend) + outL * blend`, same for R
  - Write final to buffer
- After loop: if airSmoothing just completed (was smoothing, now isn't), call updateFilters() for final target value

**Important implementation notes:**
- Use `signalsmith::filters::BiquadStatic<float>` for all filters. The operator() processes a single sample.
- The `highShelfDb(scaledFreq, db, octaves)` API takes scaledFreq = freq/sampleRate. Same for lowShelfDb and allpassQ.
- For cheapEnergyCrossfade: `#include <signalsmith-dsp/mix.h>`, call `signalsmith::mix::cheapEnergyCrossfade(float t, float &a, float &b)` where t goes from 1.0 to 0.0 (1.0=fully a, 0.0=fully b). Actually verify the API -- it may use a different convention. If the API takes t as 0-1 where 0=first and 1=second, adjust accordingly. The function writes the gain factors into a and b.
- Do NOT use a lowpass filter for HF absorption. This is the single most important design rule. High shelf preserves HF content above the transition, just attenuated.
- Keep allpass Q below 1.0 even at max to avoid metallic artifacts (max across all characters is 0.8 for Warm).
- Filter coefficient calculation involves transcendental functions -- only recalculate every 16 samples during smoothing to save CPU.
